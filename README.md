# Redress
Digital closet

## Development

This project is now a **Next.js (App Router) + React** app.

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Routes

- `/` (Home)
- `/closet`
- `/wishlist`
- `/outfit-builder`

Legacy URLs are supported via redirects:

- `/closet.html` → `/closet`
- `/wishlist.html` → `/wishlist`
- `/outfit-builder.html` → `/outfit-builder`
